# Flutter Development Setup Guide

## Environment Requirements
- **Operating System:** Windows 10 or later, Linux, or macOS
- **Disk Space:** 1.64 GB (does not include disk space for IDE/tools)
- **Tools:** Git (for version control)

## Flutter SDK Installation
1. **Download the SDK**
   - Visit the [Flutter SDK releases page](https://flutter.dev/docs/development/tools/sdk/releases) and download the appropriate version for your operating system.

2. **Extract the SDK**
   - Extract the downloaded zip file to a desired location. For example, `C:\src\flutter` for Windows or `~/flutter` for macOS/Linux.

3. **Update your PATH**
   - Add the Flutter bin directory in your system environment variable.


## IDE Installation
- **Visual Studio Code** or **Android Studio**
- Install the Flutter and Dart plugins:
  - For VS Code: Go to Extensions and search for "Flutter" and "Dart".
  - For Android Studio: Go to Preferences > Plugins and search for "Flutter".


## Dependency Information
- **Flutter:** Ensure that you are using a compatible version with your project.
- **Dart:** Flutter comes with Dart SDK built-in.
- **Firebase for Flutter:** If your project uses Firebase, add the necessary Firebase packages in your `pubspec.yaml` file.

## Summary
Follow the above instructions and you will be ready to develop Flutter applications. For more details, check the [official Flutter documentation](https://flutter.dev/docs).
